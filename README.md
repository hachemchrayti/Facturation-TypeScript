# Facturation-TypeScript
Projet de Facturation développé en TypeScript

Ce projet permet de  Creér des factures/devis
![image](https://user-images.githubusercontent.com/7089605/111195151-3f765e80-85bc-11eb-964f-5b8a92be3ea0.png)

il permet aussi à stocker les devis/factures avec le local storage du navigateur sous format de array de html(l html contient une page de données saisies par le user)
